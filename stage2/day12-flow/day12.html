有时候，类似下面这种情况，浮动元素被一个 div 包裹着，如果使用 clear 属性进行清除时
会发现红色区域并不能把浮动元素包裹着，这是因为 float 元素脱离了流式布局

this is headline 1

float

This is a cyan paragraph

This is a paragraph with clear

This is other paragraph in other div

可以通过下面的方法来解决这个问题：

.wrapper::after {
content: "";
clear: both;
display: block;
}

也可以通过 overflow 属性解决，给 wrapper 选择器添加 overflow：auto。

.wrapper {
background-color: red;
color: #fff;
padding: 10px;
overflow: auto;
}
